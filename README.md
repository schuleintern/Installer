# SchuleIntern - installer


## How To Use

- Download Repro
- Move "www" to your Webserver
- open YOUR_URL/install





## Development


### Project init
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run release
```

### PHP Modification

at ./install.php
NOT ./www/install/install.php !!!
